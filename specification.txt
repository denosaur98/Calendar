Приложение сделано с помощью библиотеки moment.js, установите её (npm/yarn i moment).
Backend приложения реализован на json-server на localhost:5000, также установите его (npm/yarn install json-server).
При запуске приложения используйте два терминала:
1. первый для приложения(npm start)
2. второй для сервера(json-server -p 5000 --watch db.json)
